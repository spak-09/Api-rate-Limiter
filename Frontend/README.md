# RateShield Frontend

Modern dashboard for RateShield built with React, Vite, Tailwind CSS, Zustand, Axios, Socket.IO, and Recharts.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```

## Notes

- API requests use `http://localhost:5000`
- JWT token is stored in localStorage and attached automatically
- Dark theme with glassmorphism and SaaS dashboard styling
